# Console
Simple c++ Attach console for Windows

Of course, it is also available in DLL programs    

## Example

```cpp
    Utils::AttachConsole();
    Utils::InitConsole();
    Utils::ConsolePrint("Hello, I'm zhitkur\n");

    system("pause");
    Utils::DetachConsole();
```
